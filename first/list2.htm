Blocked Elements

- which start from new line and covers whole width of screen

    Examples:
  1. ul
  2. ol
  3. h1 to h6
  4. pre
  5. address
  6. div
  7. section

Inline Elements

- which starts from where prev elements ends

    Examples:
  • img
  • video
  • audio
  • span
  • a
  • button
  • bold
